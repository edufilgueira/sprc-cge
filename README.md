## Informações

* Ruby 2.4.9
* Rails 5.0.7.2
* PostgreSQL

## Instalação


  1) Instalar RVM

  2) rvm use --create 2.4.9@sprc

  3) gem install bundler

  4) bundle install

  5) rspec (para rodar os testes)

  6) rails s
